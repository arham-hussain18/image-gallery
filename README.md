# 📸 Responsive Image Gallery using HTML, CSS & JavaScript

A modern and responsive Image Gallery built using **HTML5**, **CSS3**, and **JavaScript**. The project features a clean user interface, Lightbox image preview, Previous/Next navigation, hover animations, smooth transitions, and a fully responsive layout for desktop, tablet, and mobile devices.

---

## ✨ Features

- 📱 Fully Responsive Design
- 🖼️ Modern Image Gallery Layout
- 🔍 Lightbox Image Preview
- ⬅️ Previous & Next Navigation
- 🎨 Hover Effects
- ✨ Smooth CSS Transitions
- 🚀 Fast Loading
- 💻 Cross-Browser Compatible
- 🧩 Clean & Well-Structured Code
- 📂 Easy to Customize
